# Python Question Answering Chatbot

## Description
This project is a simple Question Answering chatbot built using Streamlit and the Hugging Face Transformers library. The chatbot answers questions based on predefined Python notes using the DistilBERT model.

## Features
- Ask questions about Python
- Uses DistilBERT Question Answering model
- Interactive Streamlit interface
- Fast and lightweight

## Technologies Used
- Python
- Streamlit
- Hugging Face Transformers
- PyTorch

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py
